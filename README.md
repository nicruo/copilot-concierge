# Copilot Concierge

<img width="802" height="351" alt="image" src="https://github.com/user-attachments/assets/f0179c50-089a-40b2-a889-b8ea641765cb" />


Ready to launch Copilot Agent to assist your users in utilizing Copilot Chat

## Why?

Employees need fast, trusted guidance
- Copilot capabilities evolve quickly; guessing creates risk
- People want “how do I…” workflows, not long docs
- Consistent, approved messaging reduces confusion and support load

## Mission

- Explain what Copilot Chat is and what it can do
- Coach on effective prompting
- Point to Microsoft Learn for credibility and depth
- Flag privacy & data access basics when work data is involved
- Fully custom to Organization guidance and needs

## How

- Follow the guide to Build an Agent with Microsoft 365 Copilot: [Build Agents with Microsoft 365 Copilot | Microsoft Learn](https://learn.microsoft.com/en-us/microsoft-365-copilot/extensibility/agent-builder-build-agents)

- Copy the instructions (and suggested prompts) from this project
- Download the zip package of the created Agent
- Go to Microsoft Admin Center and publish your Agent: [Microsoft 365 admin center](https://admin.microsoft.com)
- Pin your Agent
